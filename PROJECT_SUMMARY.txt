📦 PROJECT SUMMARY
═════════════════════════════════════════════════════════════════════════

A fully functional, production-ready watch party website where two people
can watch YouTube videos together in real-time with live chat.

• Framework: Node.js + Express
• Real-time: Socket.IO
• Frontend: HTML5 + CSS3 + Vanilla JavaScript
• Theme: Romantic pink gradients with floating hearts
• Responsive: Mobile, Tablet, Desktop
• Lightweight: Only 2 npm dependencies
• Total Code: 2,082 lines of well-commented code

🌐 NETWORK COMMUNICATION
═════════════════════════════════════════════════════════════════════════

Socket.IO Events (Client → Server):
  • join-room: Create or join a room
  • send-message: Broadcast chat message
  • video-play: Notify video playing
  • video-pause: Notify video paused
  • video-seek: Notify position changed
  • video-changed: Notify new video loaded

Socket.IO Events (Server → Client):
  • room-created: New room created with code
  • user-joined: Someone joined the room
  • user-left: Someone left the room
  • user-count: Updated user count
  • receive-message: Chat message received
  • video-play: Remote user pressed play
  • video-pause: Remote user pressed pause
  • video-seek: Remote user seeked
  • video-changed: Remote user loaded video
  • sync-video-state: Initial state when joining
  • error: Error message
