🎯 HOW IT WORKS:
─────────────────

CREATOR FLOW:
  1. Enter your name
  2. Click "Create Room"
  3. Get room code & link
  4. Share link with friend

FRIEND FLOW:
  1. Enter your name
  2. Click "Join Room"
  3. Enter room code
  4. Join the room

WATCH VIDEOS:
  1. Paste YouTube URL
  2. Click "Load Video"
  3. Press Play (auto-syncs)
  4. Chat in real-time

🐛 TROUBLESHOOTING:
────────────────────

❌ "Port 3000 already in use"
   → Try: PORT=3001 npm start

❌ "Cannot find module"
   → Run: npm install

❌ "Video not syncing"
   → Refresh page and rejoin room
   → Check YouTube URL format

❌ "Messages not appearing"
   → Check browser console (F12)
   → Verify Socket.IO connection

❌ "Can't access from another device"
   → Check firewall settings
   → Use correct IP address
   → Ensure device is on same network


💡 PRO TIPS:
────────────

1. Use Chrome/Firefox for best experience
2. Stable internet = better video sync
3. Copy room link is easiest way to share
4. Allow 1-2 seconds for initial video sync
5. Refresh if experiencing desync issues
6. Check browser console for error details
