## ⚡ Quick Start (3 steps)

### 1️⃣ Install dependencies
```bash
npm install
```

### 2️⃣ Start the server
```bash
npm start
```

### 3️⃣ Open in browser
```
http://localhost:3000
```

**That's it!** Your watch party is ready! 🎉

---

## 🚀 How It Works

**User A (Creator):**
1. Enter name → "Create Room"
2. Get room code & link
3. Share with User B

**User B (Friend):**
1. Enter name & room code → "Join Room"
2. Connected! 💕

**Watch Videos Together:**
1. Paste YouTube URL
2. Load video
3. Press Play (auto-syncs!)
4. Chat in real-time

## 🐛 Troubleshooting

**Port already in use?**
```bash
PORT=3001 npm start
```

**Can't access from other device?**
- Check firewall
- Use correct IP address
- Ensure same network

**Videos not syncing?**
- Refresh and rejoin
- Check YouTube URL
